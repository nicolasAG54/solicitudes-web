# Frontend - solicitudes-web

Este es el frontend desarrollado en **Angular moderno (standalone + signals)** para el proyecto fullstack con backend en **ASP.NET Core 7** y base de datos **SQLite**.

## 🛠 Tecnologías

- Angular 17+ (Standalone Components, Signals)
- TypeScript
- MATERIAL angular / SCSS (si aplica)
- RxJS
- HTTPClient para consumir API REST del backend

## 🚀 Requisitos

- Node.js >= 20
- npm >= 9
- Angular CLI >= 17
- URL del repositorio donde encontrar el backend https://github.com/nicolasAG54/solicitudes-service

## ⚡ Instalación

Clona el repositorio y ejecuta:

```bash
npm install
